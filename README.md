# rick.treur.be

Source files for [rick.treur.be](http://rick.treur.be), a memorial site made with [Jekyll](http://jekyllrb.com/). This project contains the required source files for the pages and some tools for easy development and writing.

## Build
1. Install Jekyll
2. Run `jekyll build`
